# Installation

Clone repository:

```bash
git clone https://github.com/username/project-name.git
```

Masuk ke folder project:

```bash
cd project-name
```

Install dependency:

```bash
composer install
```

Copy environment file:

```bash
cp .env.example .env
```

Generate application key:

```bash
php artisan key:generate
```

Run migration:

```bash
php artisan migrate
```

Jalankan server:

```bash
php artisan serve
```
