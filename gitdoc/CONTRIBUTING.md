# Contributing Guide

Terima kasih telah berkontribusi pada project ini.
Dokumen ini menjelaskan proses kontribusi agar pengembangan tetap terstruktur.

---

# Cara Berkontribusi

## 1. Fork Repository

Fork repository ini ke akun GitHub Anda.

---

## 2. Clone Repository

```bash
git clone https://github.com/username/project-name.git
```

---

## 3. Buat Branch Baru

Gunakan branch baru untuk setiap perubahan.

```bash
git checkout -b feature/login-api
```

---

## 4. Lakukan Perubahan

Lakukan perubahan pada kode sesuai kebutuhan.

Pastikan kode:

- rapi
- mengikuti coding standard
- tidak merusak fitur lain

---

## 5. Commit Perubahan

Gunakan commit convention yang telah ditentukan.

Contoh:

```
feat(auth): add login endpoint
fix(cart): correct total price calculation
```

---

## 6. Push Branch

```bash
git push origin feature/login-api
```

---

## 7. Buat Pull Request

Buka repository GitHub lalu buat Pull Request.

Isi Pull Request dengan:

- deskripsi perubahan
- alasan perubahan
- screenshot (jika ada)

---

# Pull Request Checklist

- [ ] kode sudah diuji
- [ ] commit mengikuti commit convention
- [ ] tidak ada error
- [ ] dokumentasi diperbarui jika perlu
