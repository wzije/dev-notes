# Dev Notes

Repository ini berisi kumpulan **catatan, panduan, dan referensi teknis** yang digunakan selama proses pengembangan software.

Tujuan dari repository ini adalah untuk menyimpan berbagai **best practices, workflow, dan dokumentasi pengembangan** agar mudah diakses kembali oleh developer maupun tim pengembang.

---

# Purpose

Project **dev-notes** dibuat untuk:

* menyimpan catatan pengembangan software
* mendokumentasikan workflow development
* menyimpan standar coding dan workflow tim
* menjadi referensi saat mengembangkan project baru

Repository ini dapat digunakan sebagai **knowledge base developer**.

---

# Contents

Repository ini berisi berbagai dokumentasi terkait pengembangan software, antara lain:

* Git Workflow
* Commit Convention
* Branch Naming Convention
* Pull Request Guidelines
* Development Guidelines
* Best Practices Development
